Лабораторная работа №4
Умножение частоты

Цель работы

  1. Изучение процесса умножения частоты.
  2. Получение оптимального режима умножителя.

Схема работы и измерительная аппаратура

Для работы используется сменный блок ПРЕОБРАЗОВАНИЯ СИГНАЛОВ В НЕЛИНЕЙНОЙ ЦЕПИ. Схема исследуемой цепи и её описание приведены в лабораторной работе №3 (рис. 3.1) — резистивный усилительный каскад на полевом транзисторе с LC-контуром в цепи стока.

Переключатель «НАГРУЗКА» — в положение «LC», кнопка «RШ» — отжата. Источник входного сигнала — встроенный диапазонный генератор, подключённый к любому входу сумматора (например, гнёзда 1).

В работе используется сток-затворная характеристика полевого транзистора из работы №2 и соответствующие аппроксимирующие функции для расчёта спектра.

Демонстрация: спектр тока умножителя

График показывает амплитуды гармоник тока стока в зависимости от угла отсечки θ. Видно, что для каждой гармоники существует оптимальный угол отсечки, при котором её амплитуда максимальна. Используйте слайдер для выбора угла и наблюдайте изменение спектра.

Предварительный расчёт

Перед выполнением работы необходимо:

  1. Аппроксимировать сток-затворную характеристику своего стенда кусочно-линейной и кусочно-параболической функциями (из работы №2).
  2. Рассчитать амплитудную характеристику умножителя одним из двух способов по указанию преподавателя:
    • Способ 1 — при постоянном максимальном токе (iмакс = const): метод угла отсечки с коэффициентами γn(1)(θ) для кусочно-линейной аппроксимации.
    • Способ 2 — при постоянной амплитуде входного сигнала (Umвх = 1 В): метод коэффициентов γn(2)(θ) для кусочно-параболической аппроксимации. ЕСМ изменять в пределах −(1÷6) В, номер гармоники n = 2÷4.

Задание

Исследуйте работу умножителя частоты при постоянной амплитуде импульса тока стока или при постоянной амплитуде входного сигнала (по указанию преподавателя).

Порядок выполнения эксперимента

1

Настройка в резонанс

Изменяйте частоту встроенного генератора в диапазоне 12–16 кГц при UВХ ≈ 0,5 В до максимума напряжения на КТ 2 или максимума показания микроамперметра. Резонансную частоту f0 внести в таблицу.

Частота входного сигнала умножителя: fВХ = f0 / n, где n — кратность умножения. Точное значение fВХ определяется по максимуму выходного сигнала при подстройке частоты генератора вблизи расчётного значения.

2

Способ 1: амплитудная характеристика при iмакс = const

Условие: максимальное напряжение на затворе постоянно равно нулю, то есть ЕСМ = −UmВХ. При этом амплитуда импульса тока постоянна, но меняется угол отсечки θ и спектральный состав тока.

Сначала устанавливать ЕСМ, затем UВХ. Перед каждым отсчётом подстраивать частоту генератора согласно п. 1.

n = … ;   f0 = … кГц ;   fВХ = f0/n = … кГц
UВХ, В 1,01,52,02,5…
UmВХ, В
ЕСМ, В
UВЫХ, В
Iсn, мА

Амплитудный ток n-й гармоники: Iсn = UВЫХ / RЭО, где RЭО = 1 кОм — эквивалентное сопротивление контура на резонансе. Найти точку максимума UВЫХ — оптимальный режим.

3

Способ 2: амплитудная характеристика при UВХ = const

Установить UВХ = 1 В, частота fВХ = f0/n. Изменять ЕСМ от −1,0 до −6,0 В с шагом 0,5 В, измеряя UВЫХ на КТ 2.

n = … ;   f0 = … кГц ;   fВХ = … кГц ;   UВХ = 1 В
ЕСМ, В −1,0−1,5−2,0−2,5−3,0…
UВЫХ, В
Iсn, мА

Найти точное значение ЕСМ, при котором UВЫХ = UВЫХmax. Построить график Iсn = φ(ЕСМ).

4

Временные диаграммы и спектры оптимального режима

При найденных оптимальных значениях ЕСМ и UВХ зафиксировать (с сохранением масштаба по оси времени):

  • Входное напряжение uВХ(t) — гнездо КТ 1, и его спектр.
  • Ток стока iС(t) — переключатель «R», гнездо КТ 2, и его спектр. По форме импульса определить угол отсечки θ.
  • Выходное напряжение uВЫХ(t) при LC-нагрузке — гнездо КТ 2, и его спектр: отдельно при RШ выключен и RШ включён.

Угол отсечки θ определяется из соотношения: θ = 180° × (ширина основания импульса тока) / (период колебания).

5

Умножение на другую гармонику

Не изменяя ЕСМ и UВХ, настроить генератор на получение гармоники с более высоким номером (n = 3÷6, по указанию преподавателя). Небольшой подстройкой ЕСМ добиться максимума UВЫХ. Зафиксировать осциллограммы и спектры uВХ(t), iС(t) и uВЫХ(t) при выключенном RШ.
